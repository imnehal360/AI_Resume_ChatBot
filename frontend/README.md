# AI Resume ChatBot - Frontend

## Features Implemented
- **Landing Page**: Modern UI with calls to action.
- **Authentication**: Login and Register (JWT based).
- **Core Chat**: Real-time resume building via chat interface.
- **Resume Preview**: Live preview of the resume as it's built.
- **Dashboard**:
  - **Job Recommendations**: Personalized job feed with match scores.
  - **ATS Scanner**: Analyze job descriptions against your resume.

## Tech Stack
- React (Vite)
- Tailwind CSS (Custom "Light Grey/White/Black" theme)
- Axios (API Client)
- Lucide React (Icons)

## Setup
1. `npm install`
2. `npm run dev` (Starts on port 5173 usually)
